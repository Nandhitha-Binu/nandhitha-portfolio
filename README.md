# 🌐 Nandhitha Binu – Portfolio

My personal Cloud & DevOps portfolio showcasing my experience, projects, technical skills, and certifications.

## 🚀 Live Website

🌍 https://nandhitha-portfolio.vercel.app/

## 📌 About

This portfolio highlights my journey as a Cloud & DevOps Engineer with hands-on experience in:

- Linux Administration (RHEL 9)
- AWS Cloud Services
- Docker
- Jenkins
- Terraform
- Ansible
- Monitoring with Prometheus & Grafana

## 🛠️ Built With

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Vercel

## 📂 Features

- Responsive design
- Dark / Light mode
- Professional resume
- Skills & Certifications
- Projects showcase
- Contact section

## 📦 Run Locally

```bash
git clone https://github.com/Nandhitha-Binu/nandhitha-portfolio.git
cd nandhitha-portfolio

npm install
npm run dev
```
